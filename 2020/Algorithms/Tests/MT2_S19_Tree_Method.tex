\section{Standard 18- Tree Method}
\begin{required}
Using the tree method, find a suitable function $f(n)$ such that $T(n) \in \Theta(f(n))$. Show all work. [\textbf{Note:} You may assume  without loss of generality that $n$ is a power of $3$; that is, $n = 3^{k}$ for some integer $k \geq 0$.]

\begin{align*}
T(n) &= \begin{cases} 
7 & : n < 3, \\
4T(n/3) + n & : n \geq 3.
\end{cases}
\end{align*}
\end{required}

\begin{proof}[Answer]
%Your answer here
We hit a base case when $n=3^k < 3$. Solving for $k$, we have that $n/3^h < 3$. As $n=3^k$, we hit a base case when $h=k$. That is we have k levels to the recursion tree, indexed from 0, 1, ... , k-1. In particular we note that, $k = \log_3(n)$.

Now, at each level $i$ of the tree, we have $4^i$ nodes. The non-recursive work done at each level is $n/3^i$. So, total work done at each level is $n(\frac{4}{3})^i$. Thus: 

$\sum_{i=0}^{k-1} n(\frac{4}{3})^i$

$n\sum_{i=0}^{k-1} (\frac{4}{3})^i$

$n(\frac{1-(\frac{4}{3})^k}{1-\frac{4}{3}})$

$-3n(1-\frac{4^k}{3^k}) = -3n + 3(4^{\log_3(n)})$

$T(n) =  3(4^{\log_3(n)})-3n$

So $T(n) \in \Theta(4^{\log(n)})$
\end{proof}
